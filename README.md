# QPICK

- Стек технологий:
  TypeScript, SCSS, React, Redux

- Пакетный менеджер:
  Yarn

- Сборщик проекта:
  Vite

- Дополнительные библиотеки:
  i18next, react-js-loader, redux-persist

- Команды для запуска:

  ```
  yarn install
  ```

  ```
  yarn start
  ```
